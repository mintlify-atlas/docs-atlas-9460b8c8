---
title: Update Environment
api: PUT /v1/admin/environments/:id
description: Update an existing environment
---

## Authentication

Requires a token with `environments:write` permission.

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the environment to update.
</ParamField>

## Request Body

At least one field must be provided.

<ParamField body="label" type="string">
  New environment label (max 255 characters).
</ParamField>

<ParamField body="region" type="string">
  New region identifier (max 100 characters).
</ParamField>

<ParamField body="api_base_url" type="string">
  New base URL (max 500 characters). Must be a valid URI.
</ParamField>

<ParamField body="api_key" type="string">
  New API key (max 512 characters).
</ParamField>

## Response

<ResponseField name="data" type="object">
  The updated environment object.
  
  <ResponseField name="id" type="string">
    Environment identifier (unchanged).
  </ResponseField>
  
  <ResponseField name="label" type="string">
    Updated environment label.
  </ResponseField>
  
  <ResponseField name="region" type="string">
    Updated region identifier.
  </ResponseField>
  
  <ResponseField name="api_base_url" type="string">
    Updated base URL.
  </ResponseField>
  
  <ResponseField name="api_key" type="string">
    Updated API key.
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    ISO 8601 timestamp of creation.
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    ISO 8601 timestamp of last update.
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X PUT https://api.weaponx.io/v1/admin/environments/production-us \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "label": "Production US East (Primary)",
    "region": "us-east-1a"
  }'
```

```json
{
  "data": {
    "id": "production-us",
    "label": "Production US East (Primary)",
```

## Error Responses

```json
{
  "error": "Validation Error",
  "details": [
    {
      "message": "\"api_base_url\" must be a valid uri",
      "path": ["api_base_url"]
    }
  ]
}
```

### 401 Unauthorized

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```

### 403 Forbidden

```json
{
  "error": "Forbidden",
  "message": "Insufficient permissions: environments:write required"
}
```

### 404 Not Found

```json

  "message": "Environment not found"
```
