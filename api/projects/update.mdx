---
title: Update Project
api: PUT /v1/admin/projects/:reference
description: Update an existing project
---

## Authentication

Requires a token with `projects:write` permission.

## Path Parameters

<ParamField path="reference" type="string" required>
  The unique reference identifier of the project to update.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  New project name (max 255 characters).
</ParamField>

## Response

<ResponseField name="data" type="object">
  The updated project object.
  
  <ResponseField name="reference" type="string">
    Project reference identifier (unchanged).
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Updated project name.
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    ISO 8601 timestamp of creation.
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    ISO 8601 timestamp of last update.
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X PUT https://api.weaponx.io/v1/admin/projects/my-app \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Updated Application"
  }'
```

## Example Response

```json
{
  "data": {
    "reference": "my-app",
    "name": "My Updated Application",
    "created_at": "2026-03-05T10:30:00Z",
    "updated_at": "2026-03-05T11:45:00Z"
  }
}
```

## Error Responses

### 400 Bad Request

```json
{
  "error": "Validation Error",
  "details": [
    {
      "message": "\"name\" is required",
      "path": ["name"]
    }
  ]
}
```

### 401 Unauthorized

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```

### 403 Forbidden

```json
{
  "error": "Forbidden",
  "message": "Insufficient permissions: projects:write required"
}
```

### 404 Not Found

```json
{
  "error": "Not Found",
  "message": "Project not found"
}
```
