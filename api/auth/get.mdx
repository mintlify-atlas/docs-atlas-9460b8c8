---
title: Get Authentication Token
api: GET /v1/admin/authentications/:id
description: Retrieve a specific authentication token by ID
---

## Authentication

Requires a token with `authentications:read` permission.

## Path Parameters

<ParamField path="id" type="string" required>
  The unique UUID of the authentication token.
</ParamField>

## Response

<ResponseField name="data" type="object">
  The authentication token object.
  
  <ResponseField name="id" type="string">
    Unique token ID (UUID).
  </ResponseField>
  
  <ResponseField name="project_reference" type="string">
    Project reference.
  </ResponseField>
  
  <ResponseField name="role_id" type="string">
    Associated role ID.
  </ResponseField>
  
  <ResponseField name="secret_key" type="string">
    The token secret.
  </ResponseField>
  
  <ResponseField name="email" type="string">
    Associated email.
  </ResponseField>
  
  <ResponseField name="description" type="string">
    Token description.
  </ResponseField>
  
  <ResponseField name="is_active" type="boolean">
    Active status.
  </ResponseField>
  
  <ResponseField name="expiration_date" type="string | null">
    Expiration date or null.
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    ISO 8601 timestamp of creation.
  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    ISO 8601 timestamp of last update.
  </ResponseField>
  
  <ResponseField name="removed_at" type="string | null">
    Soft deletion timestamp or null.
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET https://api.weaponx.io/v1/admin/authentications/950e8400-e29b-41d4-a716-446655440004 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Error Responses

### 401 Unauthorized

```json
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```

### 403 Forbidden

```json
{
  "error": "Forbidden",
  "message": "Insufficient permissions: authentications:read required"
}
```

### 404 Not Found

```json
{
  "error": "Not Found",
  "message": "Authentication token not found"
}
```
